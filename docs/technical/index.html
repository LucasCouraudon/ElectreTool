    ELECTRE Decision Support Tool

    Description

    Application web d'aide à la décision multicritère implémentant les méthodes ELECTRE II et ELECTRE Iv. L'outil permet de modéliser une matrice de performance, de gérer des poids normalisés et de configurer des seuils spécifiques de concordance, discordance et veto.

    Architecture Technique

    L'application applique les principes de séparation des préoccupations :

    • src/main.ts : Orchestrateur global et point d'entrée de l'application.
    • src/matrix.ts : Gestionnaire de l'IHM de la matrice et des calculs de normalisation.
    • src/persistence.ts : Couche de persistance synchrone (localStorage) et gestion de l'état.
    • src/history.ts : Contrôleur du cycle de vie des versions et du volet d'historique.

    Installation et Lancement

    Prérequis

    • Node.js
    • Gestionnaire de paquets npm

    Commandes de cycle de vie

    # 1. Installation des dépendances (Vite, TypeScript, TypeDoc)
    npm install

    # 2. Lancement du serveur de développement local
    npm run dev

    # 3. Compilation de production
    npm run
